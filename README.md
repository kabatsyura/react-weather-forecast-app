# Прогноз погоды для Татнефть

[React Weather Forecast App](https://kabatsyura.github.io/react-weather-forecast-app/)

![Приложение для прогноза погоды](for_readme/tatneft-app.png)



Все сборки для проекта расположены в файле Makefile.

Для запуска локально, попробуйте команду:

```
make dev
```

Чтобы запустить приложение из контейнера:

1. Сделайте билд контейнера:
```
make docker-build
```

2. Запустите его:
```
make docker-run
```

3. Далее, перейдите на localhost

[Localhost](http://localhost)
